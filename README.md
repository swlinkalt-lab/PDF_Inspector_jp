# PDF Inspector Business
**大量のPDFから、  
「人が確認すべきPDFだけ」を一括で抽出する  
業務向けPDFキーワード検索ツール（Windows）**

---

数十〜数百件のPDFを、  
**1件ずつ開いて確認する作業に時間を取られていませんか？**

契約書・仕様書・報告書・監査資料など、  
業務で扱うPDFが増えるほど、  
「読む作業」そのものがボトルネックになりがちです。

PDF Inspector Businessは、  
指定したキーワードを含むPDFだけを一括で抽出し、  
**人が目視確認すべき対象を最初から絞り込むための業務用ツール**です。

「すべて読む」作業を、  
**「読む必要のあるものだけを選ぶ」作業に置き換えることで、  
確認・レビュー・監査にかかる時間を大幅に削減**します。

> 💡 これは「PDFを読むツール」ではありません  
> **「読む必要のあるPDFだけを選ぶためのツール」です**

## ⚠ ご注意ください（重要）

PDF Inspector Businessは、  
**法務文書や契約書の正確な全文比較や、  
完全一致を保証するツールではありません。**

あくまで  
**「人が確認すべきPDFを効率よく選び出す」ための支援ツール**です。  
最終的な判断・確認は、**必ず人の目で行ってください。**

この前提を理解した上で使うことで、  
確認作業の負担を大きく減らすことができます。

---

## ✅ このツールでできること

- PDFフォルダを丸ごとスキャン  
  → **1件ずつPDFを開く作業が不要になります**
- 指定キーワードの一括検索（AND / OR対応）  
  → **読む必要のある資料だけがすぐ分かります**
- 条件に合ったPDFの抽出・整理  
  → **確認対象をそのままチームや上司に渡せます**

> 確認作業に追われる時間を減らし、  
> **判断・対応・次のアクションに集中できるようになります。**

## 📂 対応PDF・注意点 / Supported PDF Types & Notes

### 対応PDF

- PC作成のテキストベースPDF  
- 選択・コピー可能な横書き文書（日本語／英語）  

### 注意が必要なPDF

- 縦書きや手書き文字入りPDF  
- 画像スキャンのみのPDF  
- 小さすぎる文字や不鮮明なPDF  

> テキストが少ない場合はOCR補助を実行しますが、**完全認識は保証できません**。

## 🏃 使い方 / How to Use

1. PDF Inspector Business（exe）を起動  
2. PDFフォルダを選択（最大300件、推奨100件以内）  
3. 検索キーワードを入力（最大5語）  
4. AND / OR を選択  
5. **事前確認モード**を選ぶか実行  
6. 実行ボタンで処理開始  

> Preview Mode（dry-run）で件数確認後に実行すると安心です。
